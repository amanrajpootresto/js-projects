# PRODUCT FEATURE LIST

Who will use this application?
``` For my personal expense tracking ```

What problem does it solve?
``` I cna't remember where i spend so it will help me track the same in  different format ```
- Graph, Pie Chart, Spending Ration
- Monitor my habits
- More detaield then any other payment apps
- Also track my offline spending

What must a user be able to do?
``` Add an expense ( offline + online) + Context of spending + Date/Time + Amount + Location + Need/Priority set ( Phase 2 ), Filter/Sort and Track events, add income ```

What information must be stored?
``` All expenses, Past spending, Transaction ( Mode, Amount, Reason, Priority, Date/Time + Location), income ```

What calculations are needed?
``` 
    - Total spending
    - Classification based on category
    - Based on priority/Need
    - Most spending
    - Repetitive spending
    - Savings
    - Habits
```

What could go wrong?
``` ns ```
